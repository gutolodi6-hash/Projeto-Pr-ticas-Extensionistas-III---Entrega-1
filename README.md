# Projeto Práticas Extensionistas III - Entrega 1

Este repositório contém a documentação e modelagem do sistema de informação desenvolvido para a disciplina de Práticas Extensionistas III.

## Descrição do Projeto
O sistema escolhido foca na **Gestão de Pedidos e Atendimento**, permitindo o cadastro de clientes, registro de vendas por funcionários e o acompanhamento do fluxo de pedidos.

##  Conteúdo da Entrega
O projeto contempla os seguintes artefatos de modelagem:

### 1. Modelagem de Banco de Dados
- **Modelo Conceitual:** Representação das entidades e relacionamentos.
- **Modelo Lógico:** Estrutura de tabelas, chaves primárias e estrangeiras.
*(Ferramenta utilizada: brModelo)*

### 2. Diagramas UML
- **Diagrama de Caso de Uso:** Interações entre Atores (Cliente/Funcionário) e o Sistema.
- **Diagrama de Classes:** Estrutura de dados e métodos das classes principais.
- **Diagrama de Atividades:** Fluxo lógico do processo de registro de pedidos.
- **Diagrama de Sequência:** Troca de mensagens entre os objetos para a funcionalidade de registrar pedido.
*(Ferramenta utilizada: Visual Paradigm / Draw.io)*
